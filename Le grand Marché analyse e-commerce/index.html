Analyse des ventes e-commerce

Objectif : Identifier les produits les plus rentables et comprendre les comportements d’achat.

Contexte

Jeu de données contenant 50 000 transactions issues d’un site e-commerce. L’entreprise souhaite améliorer son panier moyen et cibler les produits à fort potentiel.

Démarche

  • Nettoyage et structuration des données (Pandas).
  • Analyse exploratoire (corrélations, distributions, segmentation RFM).
  • Création d’indicateurs : CA par catégorie, marge, fréquence d’achat.
  • Visualisation dans Power BI pour support à la décision.

Outils utilisés

Python (Pandas, Matplotlib), Power BI, SQL.

Résultats

  • 3 catégories représentent 62 % du chiffre d'affaires total.
  • Segmentation RFM → identification de clients "Premium" (10 % du total, 35 % du CA).
  • Recommandation de ciblage marketing → potentiel +12 % CA.
