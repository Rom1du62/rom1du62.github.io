Réalisation d'un portfolio

Objectif : Mettre en ligne mon portfolio et répondre à une annonce.

Contexte

Ce projet présente les livrables attendus pour une candidature chez Aeroworld.

Démarche

  • Création d'un Mindmapping structurant le projet
  • Mise en place du portfolio sous GitHub en HTML
  • Rédaction de l'analyse du besoin
  • Réalisation d'un cahier des charges
  • Mise en place d'un Gantt
  • Rédaction d'une procédure pour création graphique sous Power BI
  • Mise à disposition d'une video explicative pour l'utilisation PowerBI

Outils utilisés

GIT et GITHUB, Visual Sudio code, freeplan (Mindmapping), LOOM(formation visuelle)

Compétences acquises

  • Commandes GIT
  • HTML5 et CSS.
  • Utilisation de loom.
  • Vulgarisation
  • Adaptabilité
