Base de données et requêtes SQL

Objectif : Structure et manipulation de données en SQL.

Contexte

Premier projet sur la manipulation d'une base de données en SQL, il m'a permis de me familiariser avec les dictionnaires de données, le principe de schéma relationnel normalisé grâce à l'outil Power Architects, le montage d'une base de données avec SQLite et enfin les premières requêtes SQL pour charger les données et interroger les tables qui composent la base de données.

Démarche

  • Mise en place d'un dicionnaire de données avec les différents types de données rencontrés.
  • Création d'un schémat relationnel normalisé avec Power Architects.
  • Création et chargement des données dans SQLite.
  • Rédaction de requêtes SQL.
  • Restitution dans un support de présentation.

Outils utilisés

SQLite, PowerArchitects.

Compétences acquises

  • Application des normes 3NF.
  • Power Architects pour modélisation d'un schéma relationnel.
  • SQLite pour la création d'une base de données avec deux tables.
  • Language SQL pour manipulation et interrogation des données.
