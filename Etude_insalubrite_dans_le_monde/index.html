Etude sur l'eau dans le monde

Objectif : Mettre en place un dashboard permettant d'identifier à traver le monde, des zones géographique ou une société pourrait intervenir en toute sécurité.

Contexte

Second projet sur PowerBi, il s'agit à partir des données de la DWFA (Drinking Water For All) de créer un tableau de bord pour identifier les pays qui rencontrent le plus de difficulté d'accès à l'au potable, et d'identifier ou les efforts peuvent être concentrés en toute sécurité.
Le dashboard devait se décliner en 3 vues : Une vue Mondiale, une vue continentale et une vue locale.

Démarche

  • Création des blue print avec les différents scénarios
  • Mise en place des mesures adéquats.
  • Création du dashboard.
  • Support de présentation.

Outils utilisés

Power BI, PowerQuery, Langage Dax.

Compétences acquises

  • Utilisation du blueprint
  • Developpement des compétences sous DAX
  • Familiarisation avec l'utilisation de Power Query
